A Nodemailer egy nyílt forráskódú NPM-csomag, amely segít leegyszerűsíteni az e-mailek küldését SMTP-n keresztül a Node.js alkalmazásban.
A Nodemailer modul megkönnyíti az e-mailek küldését a számítógépről.
A Nodemailer modul az npm segítségével tölthető le és telepíthető:
npm install nodemailer
